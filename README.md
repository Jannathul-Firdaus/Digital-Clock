# Digital Clock

A modern and responsive Digital Clock built using **React** and **Vite**. The application displays the current time in real time with a beautiful user interface and updates every second using React Hooks.

## Features

- ⏰ Real-time digital clock
- 📅 Current date display
- 🌞 AM/PM indicator
- ⚛️ Built with React Hooks (`useState` & `useEffect`)
- 🎨 Modern gradient UI
- 📱 Responsive design
- ⚡ Fast development using Vite

## Technologies Used

- React
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3

## Learning Outcomes

- React Components
- State Management
- useEffect Hook
- Date & Time Handling
- Responsive UI Design
